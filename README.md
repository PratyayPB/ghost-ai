## Canvas Storage & Autosave

The collaborative canvas automatically saves your project state securely to the cloud.

- **Vercel Blob**: Used to store the actual canvas JSON data containing your nodes and edges.
- **Prisma Database**: Stores the metadata for each project, including the secure `canvasBlobUrl` pointing to your Vercel Blob storage.

Your canvas state is automatically debounced and saved after 5 seconds of inactivity. When you open a project, the canvas fetches its latest state from the blob storage only if the real-time room is empty.
